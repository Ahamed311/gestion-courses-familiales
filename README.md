# Application de gestion des courses familiales

## Contexte
Une famille souhaite numériser la gestion de ses courses afin de mieux comprendre
où part son argent et quels produits sont les plus consommés.

## Objectifs
- Enregistrer les achats
- Consulter l’historique des courses
- Analyser les tendances de consommation
- Calculer le total des dépenses

## Fonctionnalités
- Ajout d’un achat (produit, prix, date)
- Historique des achats trié par date décroissante
- Analyse du produit le plus acheté (par occurrences)
- Bilan financier (montant total des dépenses)

## Architecture
Application web basée sur le modèle MVC (Modèle – Vue – Contrôleur)
